# CoinFlippr
Coin flipping simulator made in python and opens in terminal.
Some commands are:
  
  Sim <number>: flips coins until you get <number> of heads/tails in a row
  
  Calc <number>: calculates the odds of getting <number> of heads/tails in a row
  
  View more commands by typing "help"
